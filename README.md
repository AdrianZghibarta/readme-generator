# readme-generator

> Generate your project's README from package.json data.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i -g readme-generator
```

## CLI

From the command line, run:

```sh
$ readme
```

## Running tests

Install dev dependencies:

```sh
$ npm i -d && npm test
```
